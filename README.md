# Hostel Issue Tracker

A centralized web platform that enables hostel students to report maintenance issues and allows admins to assign, track, and resolve them by replacing informal complaint methods like WhatsApp groups 
and paper registers.

Built for ACE Hack — CBS Codefest 2.0 | Track: Social Impact / Sustainability

## Features
- Student registration & login
- Submit hostel issues (plumbing, electricity, cleanliness etc.)
- Real-time issue status tracking
- Admin dashboard to manage and resolve issues
- Priority-based issue categorization
- Analytics: pending vs resolved counts

---

## Tech Stack
- Backend: Python (Flask)
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Authentication: Role-based access control
